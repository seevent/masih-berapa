# Workflow: idea-orchestrator

Follow the instructions in the [idea-orchestrator skill](file:///C:/Users/Yuli%20Syarif/.gemini/config/skills/idea-orchestrator/SKILL.md).

Pipeline stages:
1. **Input Ide**: Capture the raw feature request.
2. **`grill-me`**: Ask clarifying questions to eliminate design ambiguity.
3. **`to-spec`**: Draft and validate the technical specification document.
4. **`to-ticket`**: Break the spec into actionable implementation tickets.
5. **`implement`**: Build and run empirical tests step-by-step.
6. **`code-review`**: Audit code changes against spec requirements.
7. **Output**: Deliver final feature report and walkthrough.
